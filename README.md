# c+ 
----------

**c+ alpha version**

	simulate standard container of C++ in C.

	gc              ->  Garbage Collector
    argParse        ->  get opt command line
	stdio           ->  standard stdio 
	hmap            ->  hash map        (char* / wchar_t*)   
	vector          ->  std::vector
    stack           ->  std::stack
    iterator        ->  std::iterator
    string          ->  std::string
    wstring         ->  std::wstring
